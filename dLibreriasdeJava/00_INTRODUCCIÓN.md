## LIBRERÍAS NATIVAS DE JAVA 

Las librerias nativas de Java son aquellas que que vienen por defecto en el 
(**Java Development Kit** [(JDK)](/aINTRODUCCIÓN/ConceptosBásicos/02_JDK.md)) proporcionan funcionalidades 
importantes para el desarrollo de aplicaciones Java, a continuación vamos a
ver algunas de las más importantes:

- (**01:** [(java.lang)](/dLibreriasdeJava/01_javaLang.md))
- (**02:** [(java.util)](/dLibreriasdeJava/02_javaUtil.md))
- (**03:** [(java.io)](/dLibreriasdeJava/03_javaIo.md))
- (**04:** [(java.net)](/dLibreriasdeJava/04_javaNet.md))
- (**05:** [(java.awt)](/dLibreriasdeJava/05_javaAwt.md))
- (**06:** [(java.sql)](/dLibreriasdeJava/06_javaSql.md))
- (**07:** [(java.nio)](/dLibreriasdeJava/07_javaNio.md))
- (**08:** [(java.time)](/dLibreriasdeJava/08_javaTime.md))
- (**09:** [(java.concurrent)](/dLibreriasdeJava/09_javaConcurrent.md))
